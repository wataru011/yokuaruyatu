# 🧟 ZOMBIE SHOOTER — よくある広告のやつ

上からワラワラ迫ってくるゾンビを、自動連射の銃で撃ちまくる縦型シューティングゲーム。
広告でよく見るアレです。HTML5 Canvas 製・依存ライブラリなしの 1 ファイル完結。

## 遊び方

- **移動**: マウス移動 / タッチ（ドラッグ） / `←` `→` キー（`A` `D` も可）
- **射撃**: 自動連射（撃ちっぱなし）
- 敵が自機に触れるか画面下まで到達するとゲームオーバー
- 💊 / 🔱 を拾うと 8 秒間パワーアップ（連射速度UP / 3WAYショット）
- 15 秒ごとにウェーブが進み、敵がどんどん強く・多くなる
- 紫のデカブツ（タンク）は硬いが倒すと高確率でパワーアップを落とす

## ローカルで遊ぶ

`index.html` をブラウザで開くだけ。

## GitHub Pages で公開

`main` ブランチに push すると GitHub Actions（`.github/workflows/pages.yml`）が自動でデプロイします。

初回のみ、リポジトリの **Settings → Pages → Build and deployment → Source** を
**GitHub Actions** にしておくと確実です（ワークフロー側でも自動有効化を試みます）。

公開URL: `https://<ユーザー名>.github.io/yokuaruyatu/`
