# 🍣 SUSHI INVADERS — よくあるゲーム広告のやつ

上からワラワラ迫ってくる寿司の大群を、増殖する自軍で撃ちまくる縦型2Dシューティング。
×2/×3 ゲートで軍隊が増えていく、広告でよく見るアレです。敵は全部ドット絵のお寿司。
HTML5 Canvas 製・依存ライブラリなしの 1 ファイル完結。

## 敵の寿司ネタ（7種類）

| ネタ | 特徴 |
|---|---|
| 🔴 マグロ | 標準的なにぎり |
| 🟠 サーモン | 速い |
| 🦐 エビ | ジグザグに動く |
| 🟡 タマゴ | 硬い（海苔帯装甲） |
| 🟢 ワサビ | 小さくて超高速、触れると痛い |
| 🔶 イクラ軍艦 | 大型・高耐久（たくさん出る） |
| 🟤 ウニ軍艦 | レア・超硬い・高得点 |

ボスは **王冠をかぶった特大マグロ**（5ウェーブごと、ミニオン召喚付き）。

## 遊び方

- **移動**: マウス移動 / タッチ（ドラッグ） / `←` `→` キー（`A` `D` も可）
- **射撃**: 全員が自動連射（撃ちっぱなし）
- 🚪 **ゲート**: `×2` `×3` `+10` を通ると自軍が増殖（**人数上限なし**）。`-20` `÷2` の罠ゲートに注意
  - ゲートは**撃つと数値が育つ** — マイナスゲートも撃ちまくればプラスに変わる
  - `×2` は撃ち込むと `×3` に、`÷2` もプラスに変化する
- 📦 **宝箱**: 撃って壊すと報酬（兵士追加 / 武器強化 / 連射 / 3WAY / 全滅ボム💣）
- ⭐ **レベルアップ**: 敵を倒すとXPが溜まり、レベルアップ時に3枚のカードから強化を選択
  - 兵士募集 / ダメージUP / 連射UP / マルチショット / 貫通弾 / クリティカル / ホーミングミサイル / 高速弾
  - レベルが上がると自機の見た目も進化（緑 → シアン → 金の軍団）
- 🔥 **コンボ**: 連続撃破でスコア倍率アップ。途切れる前に撃ちまくれ
- ⚠ 定期的にインベーダーの**大群ラッシュ**が来る
- 👑 **5ウェーブごとに特大マグロのボス**が出現。中段に居座ってミニオンを召喚し続けるので、撃ちまくって倒せ！撃破すると大量スコア＋兵士8人のボーナス
- ☠️ **デッドラインは画面の一番下**。寿司に突破されると兵士が減る
- 敵に触れられたり突破されると兵士が減り、**全滅でゲームオーバー**
- 12 秒ごとにウェーブが進み、敵がどんどん強く・多くなる

## ローカルで遊ぶ

`index.html` をブラウザで開くだけ。

## GitHub Pages で公開

`main` ブランチに push すると GitHub Actions（`.github/workflows/pages.yml`）が自動でデプロイします。

初回のみ、リポジトリの **Settings → Pages → Build and deployment → Source** を
**GitHub Actions** にしておくと確実です（ワークフロー側でも自動有効化を試みます）。

公開URL: `https://<ユーザー名>.github.io/yokuaruyatu/`
